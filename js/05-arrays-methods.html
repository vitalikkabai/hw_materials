Методы для работы с массивами

Array.forEach(callback) – метод, который позволяет применить функцию callback последовательно к каждому элементу массива. Мутирует исходный массив

See the Pen forEach by Vladislav Trostinsky (@trostinsky) on CodePen.



Array.map(callback) – создает новый массив, такой же длинны как и исходный. Этот метод НЕ мутирует исходный массив Элементами нового массива становятся результаты функции callback. Функция callback получает параметры:

element – поочередно каждый элемент массива,

index – индекс текущего элемента,

array – исходный массив.


Особое внимание прошу обратить на наименование параметров callback функции.

Старайтесь называть параметры согласно тому, что они хранят.

Например: если вы перебираете массив flowers – называйте первый параметр не element, а flower

See the Pen map by Vladislav Trostinsky (@trostinsky) on CodePen.



Array.filter(callback) – этот метод позволяет удалить из исходного массива значения, которые не подходят по условиям функции callback. Этот метод НЕ мутирует исходный массив. Функция callback получает те же параметры что и в map.

See the Pen filter by Vladislav Trostinsky (@trostinsky) on CodePen.



Array.sort(callback) – этот метод используется для сортировки массива. Этот метод МУТИРУЕТ исходный массив Функция callback(currentElement, prevElement) получает два параматра prevElement и currentElement.

Если currentElement больше prevElement – функция должна вернуть 1

Если currentElement меньше prevElement – функция должна вернуть -1

Если currentElement равен prevElement – функция должна вернуть 0

See the Pen sort by Vladislav Trostinsky (@trostinsky) on CodePen.



Array.find(callback) – этот метод позволяет найти элемент по конкретному признаку. Обычно используется для поиска объектов внутри массива. Функция callback получает те же параметры что и в map и должна вернуть true для подходящего элемента.

See the Pen YzXXMaq by Vladislav Trostinsky (@trostinsky) on CodePen.



Array.findIndex(callback) – этот метод похож на .indexOf() и .find(). Он ищет индекс элемента по конкретному признаку. Функция callback получает те же параметры что и в map и должна вернуть true для подходящего элемента.

See the Pen hatIndex by Vladislav Trostinsky (@trostinsky) on CodePen.



Array.every(callback) – этот метод проверяет соответствует ли каждый элемент массива определенному условию Функция callback получает те же параметры что и в map и должна вернуть true или false. Если функция вернула false хотя бы для одного элемента – Array.every(callback) тоже вернет false

See the Pen BaNNEPw by Vladislav Trostinsky (@trostinsky) on CodePen.



Array.some(callback) – этот метод првоеряет соответствует ли хотя бы один элемент массива определенному условию. Функция callback получает те же параметры что и в map и должна вернуть true или false. Если функция вернула true хотя бы для одного элемента – Array.some(callback) тоже будет равен true

See the Pen some by Vladislav Trostinsky (@trostinsky) on CodePen.



Array.reduce(callback, initialValue?) – этот метод создан для того, чтобы аккумулировать значение (то есть приводить массив к одному значению). Функция callback(accumulatedValue, element, index, array) получает параметры:

accumulatedValue – Значение, которое вернул прошлый вызов callback или первый элемент массива или initialValue

element – следующий элемент массива. Если не был указан initialValue, то при первом проходе по массиву будет равный второму элементу (array[1])

index – индекс текущего элемента.

array – исходный массив.

See the Pen reduce by Vladislav Trostinsky (@trostinsky) on CodePen.



Chaining(Чейнинг) методов массива это возможность вызывать подряд несколько методов. Это возможно, благодаря тому, что некоторые методы возвращают новый массив после выпонения.

Обычно это используется для того, чтобы выполнить несколько действий с одним и тем же массивом: отфильтровать, отсортировать, привести к одному значению. Любой набор подобный действий может заменить одна функция в методе Array.reduce(callback)
