Переменные

Переменная – псевдоним для ячейки памяти, в котором хранятся некоторые данные.

Переменные нужны для того, чтобы мы имели возможность использовать и изменять некоторые значения множество раз.

Создание переменной

Существует два основных способа создавать переменную: ключевые слова let и const.

  • let – Создает переменную, которую нельзя использовать до её объявления ниже по коду, в любой момент можно изменить, а так же удалить.
  • const – Создает переменную, которую нельзя использовать до её объявления ниже по коду и нельзя изменить.
  • Есть и третий способ создания переменной с помощью ключевого слова var, этот способ использовался в старых версиях ECMAScript до ES6.

Имена переменных

  • Переменные содержат только латинские буквы(a-zA-Z) и арабские цифры(0-9)
  • Переменные не могут начинаться с цифры
  • Переменные могут содержать символы $ и _
  • Переменные нужно называть простыми и понятными словами, показывающими, что содержит переменная
  • Для разделения нескольких слов в названии переменной нужно использовать "camelCase"

Константы

Для констант так же существует несколько простых правил

  • Если вы не планируете изменять переменную можете создавать её через const, в последствии, когда захотите ниже её изменить, нужно будет лишь указать let. Такой подход позволит вам легко ориентироваться какие переменные заданы один раз, а какие изменяются ниже в коде.
  • Если вы создаете единое и никогда неизменяемое значение для вашего приложения, называйте такие константы в верхнем регистре через _

Типы данных

Всего в Javascript существует 5 примитивных типов данных.

  • Числа
  • Строки
  • Булевые
  • undefined
  • null

Строки

Есть три способа создать строку

  • С помощью одинарных кавычек
  • С помощью двоичных кавычек
  • С помощью обратных кавычек создаются шаблонные строки.
Шаблонные строки имеют преимущество перед другими способами создания строки – в них можно подставлять переменные, переносить строки и использовать одинарные и двойные кавычки без помех

Булевые значения

Булевые значения могут быть двух видов true или false. Используются обычно для создания логики программы с помощью условных конструкций

undefined

Такое значение получает переменная, если не указать ей значение при создании. Так же при удалении переменной – она примет это значение.

null

Нет значения или нет объекта. Отличается от undefined, тем, что null – мы задаем специально, что указать что объекта нет ОСОЗНАННО

Обратите внимание, что выражение typeof null всегда возвращает строку "object"

Операторы

В JS существует множество операторов для работы с переменными и литералами. Они могут быть унарными(работают с одним значеним), бинарными(работают с двумя значениями), тернарный опеатор(он всего-лишь один)

  • Оператор присвоения = – не равно, как в математике, а присвоить значение
  • Математические операторы (+, -, /, *)
  • Операторы сравнения (===, !==, >, < и другие.)
  • delete – удалить переменную или свойство объекта
  • typeof – возвращает тип переменной
  • instanceof – используется для проверки наследования прототипа (12 занятие)

Вывод данных

  • console.log(variable) – Предпочтительный способ вывести переменную для отладки или проверки значения. Вывод происходит в специальную console, которая является частью инструментов разработки Google Chrome
  • alert(variable) – Показывает окно пользователю и блокирует выполнение дальнейшего кода. Удобно использовать для обработки ошибок, чтобы уведомить пользователя о неправильной работе или неполный данных
  • document.writeln(variable) – вставляет значение переменной напрямую в HTML. Можно использовать для того, чтобы выводить данные для пользователей. (Не рекомендуется использовать в production)

Объект Math

Для работы с математикой у JS есть встроенный объект, который реализует основные функции по вычислению

  • Math.floor(number) – округление в меньшую сторону
  • Math.trunk(number) – отбросить дробную часть
  • Math.ceil(number) – округлить в большую сторону
  • Math.round(number) – округлить как обычно
  • Math.abs(number) – вернет модуль числа
  • Math.cos(number) – косинус числа. (Доступны так же другие тригонометрические функции)
  • Math.sqrt(number) – Квадратный корень числа
  • Math.cbrt(number) – Кубический корень числа
  • Math.random() – Случайное число с плавающей точкой в диапазоне от 0 до 1
  • Math.max(...numbers) – Максимум из набора переданных чисел
  • Math.min(...numbers) – Минимум из набора переданных чисел
  • Math.pow(x, y) – Возводит x в степень y
