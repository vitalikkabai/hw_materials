Задание Basic:
Ваша основная задача в этом домашнем задании: сделать таймер обратного отсчёта.
Компонент <Timer /> Пример того что должно быть

  1. Таймер должен получать props time – время с которого начинается отсчёт
  2. У таймера должны быть кнопка start/pause. При нажатии на которую соответственно таймер либо ставится на паузу, либо запускается снова
  3. Должен быть props autostart автоматического запуска. Если указан true – таймер запускается сам. (componentDidMount может вам в этом помочь)
  4. В компонент <Timer onTick={(time) => console.log("Осталось времени: " + time)} /> можно передавать функцию onTick, которая срабатывает при каждом "тике" таймера.
  5. Добавьте props step, который позволит передать интервал обновления таймера. Например раз в 1 секунду/100 мс/2с/10с. Обратите внимание на видео – нижний таймер меняется каждые 2 секунды. Все значения времени у таймера указаны в мс
  6. Длина полоски уменьшается с движением таймера(анимацию советую сделать потом отдельно, она в этом задании НЕОБЯЗАТЕЛЬНА)

Задание Advanced:

  1. Добавьте событие onTimeEnd, передается как <Timer onTimeEnd={() => console.log("Время вышло!")} />
  2. Добавьте событие onTimeStart, передается как <Timer onTimeStart={(timeLeft) => console.log("Таймер запущен!")} />
  3. Добавьте событие onTimePause, передается как <Timer onTimePause={(timeLeft) => console.log("Таймер на паузе!")} />
  4. *Задание со звездочкой или даже с несколькими*
    Если изменяется внешний props time – ваш таймер должен отрегировать.
