Задание Basic:
На выходе у вас должен получится копомнент Contracts, который работает с логикой (state) и компонент Contract, который просто отображает номер телефона, имя и фамлиию контакта.

  1. Отобразить список контактов (можно по дизайну ориентироваться на iOS/Android контакты)
  2. Добавить поле поиска перед списком.
  3. При вводе текста в поле поиска – фильтровать по фамилии пользователя.
    (Т.е если в поле ввести "эриксон" – отобразится только контакт Маршен Эриксонян"

Так же небольшой план, который поможет сделать правильно домашнее задание

  1. Создаем проект с помощью CRA
  2. Создаем компонент Contracts. (Делаем классовый или функциональный с хуками)
  3. Добавляем в state массив contacts Учтите, contacts нужно добавить внутрь объекта state (!). Чтобы можно было обращатсья как state.contacts
  4. Пишем метод render для отрисовки данных из state. На этом этапе можно отобразить хотя бы имена
  5. Создаем компонент Contact. Делаем необходимую верстку
  6. Из props достаем необходимые данные и размещаем их в верстке (firstName, lastName, phone, gender)
  7. В компоненте Contacts – добавьте в рендере массива – компонент Contact. Вы можете передать все значения из элемента вашего массива state.contacts с помощью spread оператора. {...contact}
  8. Дорабатываем стилизацию, исправляем верстку если необходимо
  9. Добавляем поле ввода (input) перед списком контактов
  10. В state добавляем еще одно значение search
  11. Выводим в value аттрибут инпута значение из state.search
  12. Добавляем обработчик события onChange у инпута. Функция обработки события(handleSearchChange) – должна менять state.search на основании event.target.value
  13. Теперь вы можете добавить логику в функцию handleSearchChange. Вам необходимо брать исходный массив contacts и отфильтровав его с помощью функции filter – поместить в state

Задание Advanced:

  1. Добавьте иконку пола к контактам(М, Ж)
  2. Сделайте так, чтобы поиск фильтровал по имени/фамилии/номеру.
    Например: в поле поиска введено "95" – мы увидим контакты Барнея и Теодора
    Например: в поле поиска введено "он" – мы увидим Анонимуса и Эриксоняна
  3. Добавьте чекбоксы с полом(м, ж, не указан). По умолчанию все включены. При отключении чекбокса – происходит фильтрация.
    Например: указаны чекбоксы м и не указан – мы увидим Барнея, Теодора, Маршена и Анонимуса
    Например: указаны чекбоксы ж и не указан – мы увидим Лилию, Робин и Анонимуса
    Например: указаны чекбоксы м и ж – мы увидим Лилию, Робин, Барнея, Теодора, Маршена
    Например: указан чекбокс не указаны – мы увидим Анонимусы
