Домашнее задание по теме: React-Router

(!) Домашние задания нужно сдавать в виде странички на GITPAGES. Видео о том как сдавать ДЗ
Тут инструкции как это сделать с реактом


Задание Basic:
Создание простой SPA с навигацией. Сделайте 4 страницы:
главная (изображение и текст),
страница постов (возьмите из первой домашки и сделайте несколько постов),
страница с фотографиями(просто плитка изображений условный Инстаграм),
страница с контактами из второй домашки. Вверху страниц будет меню, через которое осуществляется навигация.


Задание Advanced:
Добавьте страницу одного контакта с подробной информацией о нем (пол, номер, телефон, описание, условная переписка). Попасть на эту страницу можно кликнув по контакту на странице контактов. Обратите внимание, что инфомрация должна быть актуальной, а ссылка меняться на /contacts/test.
Где test – username в вашем массиве contacts
