Домашнее задание::Занятие №3::Тема – HTML

(!) Домашне завдання потрiбно здавати у вигляди сторiнки GITPAGES

  • Создать HTML страничку.
  • Сделать таблицу: Имя, Фамилия, Телефон, Email.
  • Добавить 3-5 строк. Данные можно выдумывать.
  • Сделать форму: Имя, фамилия, пол, комментарий, выпадающий список с городами и чекбоксы с выбором языка.
  • По умолчанию выбранным должен быть какой-то пол.
  • В выпадающем списке городов по умолчанию должна быть опция "Выберите город", выбрать её после выбора города нельзя (должна быть disabled)
  • Загрузить на страничку gitpages
